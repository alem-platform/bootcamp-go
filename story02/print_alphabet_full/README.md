# print_alphabet_full

## Task

| Available                     |
| ----------------------------- |
| `github.com/alem-platform/ap` |

Write a program that prints the full alphabet followed by a newline.

The output should be as following:

```sh
go run main.go | cat -e
aAbBcCdDeEfFgGhHiIjJkKlLmMnNoOpPqQrRsStTuUvVwWxXyYzZ$
```
