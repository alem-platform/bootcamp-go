# print_arguments_index

Write a program that prints each argument received by the program along with its index in the format `index: argument`.

**Example:**

```sh
student:$ go run . salem student
0: salem
1: student
student:$ go run . "salem salem" salem
0: salem salem
1: salem
student:$
```
