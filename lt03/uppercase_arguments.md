# uppercase_arguments

Write a program that takes several strings as arguments and prints each string in uppercase.

**Example:**

```sh
student:$ go run . Hello World
HELLO
WORLD
student:$ go run . "Go" "is" "awesome"
GO
IS
AWESOME
```
