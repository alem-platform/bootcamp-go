# game_guess_number

> Тестить сложно будет

Напишите программу игру в котором нужно угадать загаданную программой цифру. Число может быть от 1 до 100.

**Example of usage:**

```sh
student:$ go run .
Guess number: 40
Lower
Guess number: 20
Higher
Guess number: salem
Guess number: 30
Lower
Guess number: 25
Match, you win!
student:$
```
