# game_guess_number

Write a program that plays a guessing game where the user needs to guess a number chosen by the program. The number can be between 1 and 100.

**Example of usage:**

```sh
student:$ go run .
Guess number: 40
Lower
Guess number: 20
Higher
Guess number: salem
Guess number: 30
Lower
Guess number: 25
Match, you win!
student:$
```
