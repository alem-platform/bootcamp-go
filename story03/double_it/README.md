## double_it

## Task

| Available                     |
| ----------------------------- |
| `github.com/alem-platform/ap` |

Write a program that scans integer `L` and then `L` numbers in a row.

- `L` - total numbers to be scanned.

Your program should output each number multiplied by two.

Examples:

```sh
go run main.go | cat -e
3
1 2 3
2 4 6$
```

```sh
go run main.go | cat -e
1
5
10$
```
